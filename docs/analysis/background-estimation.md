# Background Estimation

## Objective

Summarize the background-estimation method and its meeting-readiness status.

## Method Summary

The dominant background is QCD multijet production. The presentation-level
strategy is to constrain QCD using data control information, while SM `ttbar`
is taken from simulation. The method uses top-tag pass/fail information and
top-candidate mass sidebands to predict the QCD contribution in signal-like
regions.

| Component | Presentation role |
| --- | --- |
| Fail region | QCD-enriched control sample |
| Pass region | Signal-like region after top-tag requirements |
| Sidebands | Constrain transfer behavior outside the top-mass signal window |
| Transfer function | Maps fail-region QCD behavior into the pass region |
| SM `ttbar` | Simulation component subtracted or included according to the fit strategy |
